Наше решение: WebXR Платформа

SaaS-конструктор, позволяющий создавать AR-гиды без кода.

Работает в браузере

Никаких установок. Навел камеру на QR-код — экскурсия началась мгновенно (PWA).

No-Code редактор

Музейный сотрудник сам собирает сцену: загружает фото, 3D-модели и аудио.

AI-генерация

Встроенные нейросети сами пишут текст экскурсии и озвучивают его.

Научно-техническая новизна

  • Алгоритмы WebXR оптимизации: Уникальный метод прогрессивной загрузки «тяжелых» 3D-ассетов (GLB/USDZ), позволяющий AR работать даже на мобильном интернете 3G/4G.
  • LLM-оркестрация: Разработка конвейера (pipeline) для автоматической генерации исторически достоверного контента на основе "сырых" архивных данных.
  • Кроссплатформенный трекинг: Собственная библиотека распознавания образов (Image Tracking) на JavaScript, замещающая проприетарные решения Vuforia/8th Wall.

Импортозамещение: izi.TRAVEL, Sketchfab, Matterport.

План реализации (12 месяцев)

Этап 1 (Месяцы 1-6): MVP Платформы

  • Разработка архитектуры Backend и базы данных контента.
  • Создание No-code редактора (CMS) для загрузки медиа.
  • Реализация Web-плеера (без AR) для тестов.

  • Разработка модуля WebXR (распознавание маркеров в браузере).

  • Пилотное внедрение (Кейс: «Виртуальная деревня» / Партнерский музей).
  • Регистрация ПО в Роспатенте.
